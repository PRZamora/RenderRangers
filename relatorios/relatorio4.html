1. Introdução

Temos como objetivo técnicas de processamento de imagens, focando na equalização de histogramas, cálculo de histogramas, e operações de limiarização/binarização.

Para este relatório, tratamos de:

  • Equalização de Histogramas (Tutorial OpenCV)
  • Cálculo de Histogramas e Binarização (Cálculo de Histogramas, Operações de Threshold)

2. Fundamentos Básicos

Os conceitos fundamentais utilizados neste trabalho incluem:

  • Equalização de Histograma: Técnica para melhorar o contraste de uma imagem distribuindo de maneira uniforme a intensidade dos pixels.
  • Cálculo de Histogramas: Processo de quantificação de tons de uma imagem, para analisarmos contraste e intensidade.
  • Limiarização/Binarização: Conversão de uma imagem em tons de cinza para uma imagem binária.

3. Materiais e Métodos

Para o desenvolvimento do projeto foram utilizados os mesmos recursos das outras atividades:

  • Ambiente de desenvolvimento: CMAKE com a biblioteca OpenCV.
  • Equipamentos: Computadores do laboratório L401 e webcam para captura de imagens ao vivo

O roteiro dos experimentos seguiu as seguintes etapas:

  1. Leitura e equalização de histograma em imagens: Conversão de imagens para tons de cinza, cálculo, equalização e salvamento de histogramas.
  2. Leitura de imagem da webcam: Visualização da equalização de histograma.
  3. Estudo de binarização com equalização de histograma: Aplicação de limiarização.
  4. Equalização de histograma nas cores: Equalização dos três canais de cor de uma imagem.
